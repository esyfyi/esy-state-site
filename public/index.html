What is esy-state?

Esy-state is a lightweight Javascript framework that makes building reactive websites easier than ever before. With esy-state, users can bind HTML elements to a reactive store, meaning that any changes to the store will be reflected directly in the HTML DOM. This makes it possible to create dynamic and responsive websites with just a few lines of Javascript code.

Whether you're a seasoned developer or just starting out, [esy-state makes it easy to create websites using only HTML, CSS, and Vanilla Javascript. By eliminating the need for complex build pipelines and robust frameworks, you can focus on building great user experiences that are both fast and scalable.

So if you're looking to build high-performance websites that are easy to maintain and update, give esy-state a try today!
